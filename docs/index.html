Documentation

In order to create a notification call one of the 4 notification methods:

Notify.info(message, options?);
Notify.success(message, options?);
Notify.warning(message, options?);
Notify.error(message, options?);

Each of these accepts 2 parameters:

Parameter Type Description Example
message string The message text 
"My Notification"
options optional object Options 
{
    sticky: false,
    lifespan: 3000
{

Each method returns a new Notify.Notification().

var notification = Notify.info("My Notification");

notification.sticky();
notification.close();

var notification2 = Notify.info("Other Notification");
notification2.sticky().close();

Notification methods:

Method Description Returns
sticky Sets the notification to sticky this Notification
close Closes a notification this Notification

Notification options can be set globally.

// Use if scss variables are changed - set to match
Notify.animation = 300;
Notify.container = document.getElementById("notify-container");

// Default options for future notifications
Notify.sticky = true;
Notify.lifespan = 5000;
